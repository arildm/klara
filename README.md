# Klara

Minimalistic Python CLI task manager.

## Usage

### Create

    $ python3 klara.py create
    description: Create README
    topic: doc
    points: 1

### List

    $ python3 klara.py list
      ID Description                    Topic      Points    Created   Finished
    ---------------------------------------------------------------------------
       1 Create README                  doc             1 2017-07-12           
    ---------------------------------------------------------------------------
    Total points: 1
